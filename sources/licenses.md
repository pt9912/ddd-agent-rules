# Lizenzhinweise

## Repository-Lizenz

Das Repository-Grundgerüst und die ursprünglichen Texte werden unter der MIT-Lizenz bereitgestellt.

## Material Dritter

Füge keine Bilder, Diagramme, Texte oder Code Dritter hinzu, ohne Folgendes zu prüfen:

- Lizenz
- Anforderungen an die Quellenangabe
- Share-Alike-Anforderungen
- Anforderungen an Änderungen
- Vereinbarkeit mit der Repository-Lizenz

## Sicherer Standard

Wenn eine Quelle frei lesbar ist, aber keine ausdrückliche offene Lizenz hat:

- verlinke sie
- fasse sie in eigenen Worten zusammen
- zitiere sie
- kopiere keine wesentlichen Passagen oder Grafiken

## Creative Commons

Für CC-BY-Material:

- nenne den Urheber
- benenne das Werk
- verlinke die [CC-BY-Lizenz](https://creativecommons.org/licenses/by/4.0/)
- gib an, ob Änderungen vorgenommen wurden

Für CC-BY-SA-Material:

- wende die erforderlichen [CC-BY-SA-Bedingungen](https://creativecommons.org/licenses/by-sa/4.0/) auf Bearbeitungen an
- gehe nicht davon aus, dass MIT allein für bearbeitete Inhalte ausreicht
